Pre install:
    1. installer npm globalt
    2. installer bower globalt 
        npm install -g bower

Etter git klone:
    1. CD inn i "Enonic_Weather"
    2. Kjør npm install
    3. Kjør bower install
    4. Derretter kan man åpne index.html fra katalogen "Views/pages"
    5. Enjoy the weather :)
